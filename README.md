# Elden Ring Penance
Extreme difficulty overhaul of **Elden Ring** videogame.

## Modifications
Check out the [**MODIFICATIONS**](MODIFICATIONS.md) file to see the applied
modifications.

## Requirements
To run **Elden Ring Penance**, you need to launch the mod profile
[**`eldenring-penance.me3`**](eldenring-penance.me3) using [**Mod Engine 3**][me3_page].
